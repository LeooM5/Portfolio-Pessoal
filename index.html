My skills

HTML (HyperText Markup Language) é a linguagem base utilizada para estruturar o conteúdo de páginas web. Com o HTML, é possível organizar textos, imagens, links, tabelas e outros elementos, formando a espinha dorsal de qualquer site. Ele define o que será exibido e como os elementos se relacionam na página. Além disso, o HTML é fundamental para a acessibilidade, permitindo que pessoas com deficiência naveguem e compreendam o conteúdo.

CSS (Cascading Style Sheets) é responsável pela aparência visual das páginas web. Com o CSS, você pode definir cores, fontes, tamanhos, espaçamentos, posicionamento e criar layouts responsivos, tornando o site mais atraente e adaptável a diferentes dispositivos. O CSS também permite animações, transições e efeitos visuais avançados, além de facilitar a separação entre estrutura e apresentação, tornando o desenvolvimento mais organizado e eficiente.

SCSS é uma extensão do CSS que adiciona recursos avançados como variáveis, funções, mixins e aninhamento de regras. Isso facilita a organização e manutenção do código de estilos, tornando o desenvolvimento mais eficiente e produtivo. O SCSS permite reutilizar trechos de código, criar temas facilmente e manter grandes projetos de forma escalável, sendo amplamente utilizado em frameworks modernos como Bootstrap.

JavaScript é uma linguagem de programação que permite adicionar interatividade, animações e funcionalidades dinâmicas às páginas web. Com JavaScript, é possível validar formulários, criar menus interativos, atualizar conteúdos sem recarregar a página e muito mais. É compatível com diversas bibliotecas e frameworks, como React, Vue e Angular, e também pode ser usado no backend com Node.js, tornando-se uma das linguagens mais versáteis e populares do mundo.

Node.js é um ambiente de execução (runtime) de JavaScript assíncrono e orientado a eventos, construído sobre o motor V8 do Google Chrome. Ele permite executar código JavaScript no lado do servidor (backend), possibilitando o desenvolvimento de aplicações web escaláveis, APIs e serviços de rede de alta performance.

MySQL é um sistema de gerenciamento de banco de dados relacional (SGBD) de código aberto, baseado em SQL (Structured Query Language). Ele é amplamente utilizado para armazenar, organizar e recuperar dados de forma estruturada em tabelas, sendo uma das soluções mais populares para aplicações web e empresariais.

Git é um sistema de controle de versões que permite gerenciar o histórico de alterações em projetos de software. Com o Git, é possível trabalhar em equipe, criar diferentes versões do projeto, recuperar arquivos antigos e colaborar de forma segura e eficiente. O Git facilita o desenvolvimento distribuído, integração contínua e automação de processos, sendo essencial para projetos profissionais e open source.

Viglat

O VigLat é um sistema IoT de baixo custo para monitoramento de temperatura e umidade relativa do ar (URA) em câmaras de maturação de Queijo Minas Artesanal (QMA). Desenvolvido com Arduino, sensores DHT11 e um site web com dashboard de gráficos, ele coleta dados em tempo real, armazena em MySQL e envia alertas para variações críticas.
